# Bakery Bot — как работает бот и какие опции выполняет

## Общее описание

Бот предназначен для **контроля и напоминаний** в кофейне/пекарне. Работает в Telegram-группе: отправляет запросы отчётов по расписанию, принимает фото и видео, напоминает о невыполненных задачах.

---

## Как это работает

### 1. Расписание

Бот по расписанию отправляет в группу запросы:

| Задача | Дни | Время по умолчанию | Что нужно |
|--------|-----|-------------------|-----------|
| **Открытие** | Ежедневно | 8:00 | Подтвердить чеклист (/opening_ok) |
| **Молочка** | Вт, Чт, Сб | 13:00 | Фото + видео полки |
| **Выпечка** | Ежедневно | 20:00 | Фото витрины |
| **Закрытие** | Ежедневно | 21:00 | Подтвердить чеклист (/closing_ok) |
| **Памятка** | Ежедневно | 9:00 и 15:00 | Напоминание об обязательных работах |

### 2. Напоминания и эскалация

- **Через 15 минут** — напоминание, если задача не выполнена
- **Через 30 минут** — эскалация: сообщение «НЕ СДАНО» с перечнем недостающего

### 3. Приём контента

- **Молочка:** сначала фото полки, затем короткое видео (5–10 сек)
- **Выпечка:** только фото витрины
- Бот принимает фото/видео только в течение 30 минут после запроса

## Команды бота

### Основные (работают везде)

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие и кнопки |
| `/status` | Статус всех задач за сегодня |
| `/memo` | Памятка с обязательными работами |
| `/id` | Показать chat_id (для настройки GROUP_ID) |
| `/ping` | Проверка работы бота |
| `/keyboard` | Тест кнопок |

### Задачи (только в целевой группе)

| Команда | Действие |
|---------|----------|
| `/milk` | Запросить отчёт по молочке (фото + видео) |
| `/bakery` | Запросить фото выпечки |
| `/opening` | Запросить чеклист открытия |
| `/closing` | Запросить чеклист закрытия |
| `/opening_ok` | Подтвердить чеклист открытия |
| `/closing_ok` | Подтвердить чеклист закрытия |

### Кнопки

- **В личке:** постоянная клавиатура внизу экрана (Статус, Памятка, Молочка и т.д.)
- **В группе:** inline-кнопки под сообщением (при /start)

## Настройка (.env)

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен от @BotFather | — |
| `GROUP_ID` | ID группы (получить через /id) | -1001234567890 |
| `REMIND_AFTER_MIN` | Минуты до напоминания | 15 |
| `ESCALATE_AFTER_MIN` | Минуты до эскалации | 30 |
| `MILK_HOUR`, `MILK_MIN` | Время запроса молочки | 13:00 |
| `BAKERY_HOUR`, `BAKERY_MIN` | Время запроса выпечки | 20:00 |
| `OPENING_HOUR`, `OPENING_MIN` | Время чеклиста открытия | 8:00 |
| `CLOSING_HOUR`, `CLOSING_MIN` | Время чеклиста закрытия | 21:00 |
| `MEMO_HOUR_1`, `MEMO_MIN_1` | Первая памятка за день | 9:00 |
| `MEMO_HOUR_2`, `MEMO_MIN_2` | Вторая памятка за день | 15:00 |

## API (для мониторинга)

При запуске бота поднимается HTTP API на порту 8080:

| Endpoint | Описание |
|----------|----------|
| `GET /health` | Health check |
| `GET /ready` | Readiness probe |
| `GET /status` | Статус задач за день (JSON) |

---

## Памятка — обязательные работы

1. **Утро (открытие):** уборка зала, проверка оборудования, выкладка выпечки, кофемашина → `/opening_ok`
2. **Молочка** (Вт/Чт/Сб): фото полки + видео 5–10 сек
3. **Выпечка:** фото витрины
4. **Вечер (закрытие):** уборка, выключение оборудования, учёт выручки → `/closing_ok`
