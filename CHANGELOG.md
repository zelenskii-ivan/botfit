# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.1.0] — 2026-04-18

### Добавлено

- Команда `/help` — дублирует справку `/start`.
- Модуль `bot/status_text.py` — единое форматирование статуса для чата и inline-кнопок.
- Эндпоинт `GET /version` и поле `version` в ответе `GET /`.
- Сводка `summary.all_done` в `GET /status`.
- Задача **freezer** в JSON `GET /status` (раньше отсутствовала).
- Поля `done` и `status` у каждой задачи в `GET /status`.
- Поддержка **видеокружка** (`video_note`) для этапа видео по молочке.
- Лог при старте: `Bakery Bot v…`.

### Изменено

- Повторный ручной запрос задачи переназначает напоминание и эскалацию в планировщике (`replace_existing`), без дублирования джобов.
- Время в API: UTC с timezone вместо устаревшего `utcnow()`.
- Текст справки `/start`: заморозка, `/freezer`, видеокружок.

### Исправлено

- После сдачи фото/видео по медиа-задачам выставляется `status: done` в состоянии.
